# 护眼提醒

一个本地 Web 护眼提醒小工具。后端使用 FastAPI + SQLite，前端使用 Vue + Vite。

## 功能

- 自定义提醒间隔、休息时长和稍后提醒时长，休息时长支持分钟或秒。
- 配置和计时状态持久化到 SQLite。
- 页面内强提醒：全屏遮罩、可选循环声音、标题闪烁、浏览器通知。
- 桌面级强提醒：运行桌面伴侣后，提醒会在当前主屏弹出全屏置顶窗口。
- 支持启动、暂停、重置、开始休息、稍后提醒、跳过本次。
- 后端负责计时状态，页面刷新或短暂关闭后重新打开仍会显示真实状态。
- 页面会显示“桌面伴侣：已连接/未连接”，用于确认桌面强提醒是否真的在运行。
- 支持每天重复的“请勿打扰时段”，例如会议或午休期间不弹强提醒，到时后自动顺延。

## 启动

推荐使用一键启动脚本：

```powershell
.\start.ps1
```

如果 PowerShell 阻止脚本运行，可以使用：

```powershell
powershell -ExecutionPolicy Bypass -File .\start.ps1
```

脚本会分别打开后端、前端和桌面提醒伴侣三个 PowerShell 窗口。

手动启动：

后端：

```bash
cd backend
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

前端：

```bash
cd frontend
npm install
npm run dev
```

桌面提醒伴侣：

```bash
python desktop_reminder.py
```

打开 Vite 输出的本地地址，通常是 `http://127.0.0.1:5173`。

桌面提醒伴侣需要后端保持运行。触发提醒时，网页会创建桌面提醒任务，伴侣进程会轮询后端并在当前主屏弹出全屏置顶窗口。

点击桌面提醒伴侣里的“我知道了，开始休息”后，如果网页仍处于提醒状态，会自动进入休息倒计时。

现在计时推进由后端完成。即使网页关闭，只要后端和桌面提醒伴侣仍在运行，到点后桌面伴侣也能触发全屏提醒。

网页中的“桌面伴侣”状态来自伴侣进程心跳。启动 `desktop_reminder.py` 后通常会在几秒内显示“已连接”；关闭伴侣后会变为“未连接”。

声音提醒可以在页面设置区通过“声音提醒（蜂鸣）”开关开启或关闭；关闭后仍保留网页遮罩、标题闪烁、浏览器通知和桌面置顶提醒。

请勿打扰时段可以在页面设置区新增、删除、启用或停用。时段按每天重复处理，支持跨午夜，例如 `22:00` 到 `07:00`。如果提醒到期时处于请勿打扰时段，后端不会创建网页/桌面强提醒，而是顺延到该时段结束后再提醒。
